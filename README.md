# 🌡️ Temperature Converter Web App

A modern and responsive **Temperature Converter** built using **HTML5, CSS3, and JavaScript**. This web application allows users to convert temperatures accurately between **Celsius**, **Fahrenheit**, and **Kelvin** through a clean and user-friendly interface.

## ✨ Features

- 🌡️ Convert between Celsius, Fahrenheit, and Kelvin
- 🎨 Modern glassmorphism UI with responsive design
- ⚡ Fast and accurate temperature conversion
- 📱 Mobile-friendly and fully responsive layout
- ✅ Input validation for better user experience
- 🔄 Reset button to clear all inputs
- 🎯 Smooth animations and hover effects
- 📊 Results displayed with two decimal precision

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## 🚀 How to Run

1. Clone this repository

```bash
git clone https://github.com/your-username/temperature-converter-webapp.git
```

2. Open the project folder.

3. Run `index.html` in your preferred web browser.

No installation or backend setup is required.
